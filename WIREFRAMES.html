Interaction patterns (HTMX cheatsheet)

Tổng hợp các interaction quan trọng — implement bằng HTMX attributes thuần, không cần JS framework.

UIHTMX triggerEndpointTarget
Pagination (articles per source) click GET /sources/{n}?page=N tbody
Filter chip toggle click GET /sources/{n}?cat=X main-list
Live search keyup changed delay:300ms GET /search?q=… #results
Run progress poll every 2s GET /runs/{id}/progress closest .run-row
JSON-LD raw modal click GET /articles/{id}/jsonld #dialog
Stack reminder — FastAPI returns Jinja2 partial templates cho mỗi endpoint HTMX. Full-page request → render layout + partial. Same template, hai modes — hx-request header phân biệt.
